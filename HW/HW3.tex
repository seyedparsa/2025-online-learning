\section*{Homework 3}

Which of the following losses are mixable?
\begin{enumerate}
\item Log loss: $\lambda(x, y) = -x \log(y)-(1-x)\log(1-y)$
\item Square loss: $\lambda(x,y)=(x-y)^2$
\item Absolute Loss $\lambda(x,y)=|x-y|$
\end{enumerate}
